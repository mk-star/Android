# 공연 및 전시 기록 어플
- Android studio(kotlin) + 공연전시정보조회 서비스 API

## 🖥 프로젝트 소개
- 다양한 공연과 전시 정보를 제공하고 기록하는 공연 및 전시 기록 어플입니다.

## ⚙ 개발환경
- kotlin

## 📌 주요 기능
- 공연 및 전시의 정보를 확인할 수 있다.
- 리뷰 작성 및 수정, 삭제 기능이 있다.
- 공연 및 전시가 열리는 장소의 위치를 지도(구글맵)에서 확인할 수 있다.
